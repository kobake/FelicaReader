Constructor detail
FeliCaStatusEvent()constructor
public function FeliCaStatusEvent(type:String, object:Object, bubbles:Boolean = false, cancelable:Boolean = false)

イベントリスナーにパラメータとして渡す Event オブジェクトを作成します.

Parameters
type:String — Event.type としてアクセス可能なイベントタイプです。
 
object:Object — Event オブジェクトとして送信する対象のオブジェクトです。
 
bubbles:Boolean (default = false) — Event オブジェクトがイベントフローのバブリング段階で処理されるかどうかを判断します。
 
cancelable:Boolean (default = false) — Event オブジェクトがキャンセル可能かどうかを判断します。
Constant detail
FELICA_ACCESS_COMPLETEconstant
public static const FELICA_ACCESS_COMPLETE:String = "FeliCa Access success"

FeliCaStatusEvent.FELICA_ACCESS_COMPLETE 定数は、type プロパティ (FeliCa Access Complete イベントオブジェクト) の値を定義します.

このイベントには、次のプロパティがあります。

プロパティ 値
object 各 FeliCa アクセスの処理情報を持つクラスオブジェクトです。
bubbles false
cancelable false は、キャンセルするデフォルトの動作がないことを示します。
currentTarget イベントリスナーで Event オブジェクトをアクティブに処理しているオブジェクトです。
target 操作をキャンセルするオブジェクトへの参照です。
