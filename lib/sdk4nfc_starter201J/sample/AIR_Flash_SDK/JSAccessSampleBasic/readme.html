JSAccessSampleBasic
サンプルプログラム - README

このプログラムについて

JSAccessSampleBasic は、html上のJavaScriptからFlashファイル内の関数を呼び出し、FeliCaカードの検知およびおサイフケータイへのURL Pushを行うサンプルプログラムです。

ビルド方法

Flash Builder 4 (Flex Builder 3) および、コマンドラインでのビルド方法を説明します。

準備(共通)

  1. 本SDKの lib ディレクトリ内にある SDKforAIR_Flash_Basic.swc を、このreadmeファイルがあるディレクトリの libs ディレクトリ内にコピーしてください。

Flash Builder 4 (Flex Builder 3)

  1. 「準備(共通)」の手順を行います。
  2. このファイルがあるディレクトリを Flash Builder プロジェクト(Flex Builder 3 の場合、Flex プロジェクト)としてインポートしてください。
  3. その後、Flash Builder 4 上でのビルド、実行が可能です。

コマンドライン

  1. 「準備(共通)」の手順を行います。
  2. Flex SDK をインストールし、Flex SDK インストールディレクトリ内の bin フォルダをPATH環境変数に指定してください。
  3. build.bat ファイルを実行してください。bin-debug ディレクトリ内にswfファイルが生成されます。

実行方法

インターネットブラウザ上で生成されたswfファイルを開きます。FeliCaカードをリーダー／ライターに置き「Polling」ボタンを押すとカードのIDmを取得できます。任意のURLをプログラム上の入力欄に入力し、おサイフケータイをリーダー／ライターに置き「Push」ボタンを押すと、おサイフケータイのブラウザで指定したページを表示させることができます。

Flex SDK の fdb コマンドを使用することでデバッグメッセージ(trace)をコマンドラインに表示させることも可能です。

注意

ローカル環境でこのサンプルを実行する場合、Flash Player のセキュリティサンドボックス違反により失敗することがあります。この場合、Flash Player の設定の変更、もしくはApacheなどのウェブサーバー上に配置することで実行することが可能です。詳しくは、Flash Player のセキュリティサンドボックスの仕様についてご確認ください。
