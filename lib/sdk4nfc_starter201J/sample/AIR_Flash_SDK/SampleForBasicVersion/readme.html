SampleForBasicVersion サンプルプログラム - README

このプログラムについて

SampleForBasicVersion は本SDKの各APIを手動で実行するサンプルプログラムです。

ビルド方法

Flash Builder 4 (Flex Builder 3) および、コマンドラインでのビルド方法を説明します。

準備(共通)

  1. 本SDKの lib ディレクトリ内にある SDKforAIR_Flash_Basic.swcを、このreadmeファイルがあるディレクトリの libs ディレクトリ内にコピーしてください。

Flash Builder 4 (Flex Builder 3)

  1. 「準備(共通)」の手順を行います。
  2. このファイルがあるディレクトリを Flash Builder プロジェクト(Flex Builder 3 の場合、Flex プロジェクト)としてインポートしてください。
  3. その後、Flash Builder 4 上でのビルド、実行が可能です。

コマンドライン

  1. 「準備(共通)」の手順を行います。
  2. Flex SDK をインストールし、Flex SDK インストールディレクトリ内の bin フォルダをPATH環境変数に指定してください。
  3. build.bat ファイルを実行してください。bin-debug ディレクトリ内にswfファイルが生成されます。

実行方法

インターネットブラウザ上で生成されたswfファイルを開きます。プログラム画面左側のドロップダウンリストから機能を選んで、必要な項目を入力し「実行」ボタンを押すことでAPIを呼び出すことができます。例えば、リーダー／ライターに置かれた FeliCa カードに対してPollingを行うには以下の手順で操作します。

  1. 「NFC Proxy と認証」ボタンを押す。
  2. ドロップダウンリストから「FeliCa Access」、「リーダー／ライタ制御関係のAPI」を選択し、表示されたAPI一覧から「OpenReaderWriterAuto」を選択します。画面右側の「実行」ボタンを押します。
  3. ドロップダウンリストから「FeliCa Access」、「カードの通常コマンドAPI」を選択し、表示されたAPI一覧から「PollingAndGetCardInformation」を選択します。画面右側の「実行」ボタンを押します。
