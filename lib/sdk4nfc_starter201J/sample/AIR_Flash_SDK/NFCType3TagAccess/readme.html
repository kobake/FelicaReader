NFCType3TagAccess サンプルプログラム - README

このプログラムについて

NFCType3TagAccess はNFC Type 3 Tag へ読み込み及び書き込みを実行するサンプルプログラムです。

ビルド方法

Flash Builder 4 (Flex Builder 3) および、コマンドラインでのビルド方法を説明します。

準備(共通)

  1. 本SDKの lib ディレクトリ内にある SDKforAIR_Flash_Basic.swc または、SDKforAIR_Flash_Standard.swcを、このreadmeファイルがあるディレクトリの libs ディレクトリ内にコピーしてください。
  2. Standard版 SDK を使用している場合、src/NFCType3TagAccess.mxml をテキストエディタなどで開き、 open() 関数の引数に以下のようにライセンスキーを指定してください。("xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx" にはStandard版 SDK 購入時に発行されたライセンスキーを指定してください。)
    open(10250, "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx", "");

Flash Builder 4 (Flex Builder 3)

  1. 「準備(共通)」の手順を行います。
  2. このファイルがあるディレクトリを Flash Builder プロジェクト(Flex Builder 3 の場合、Flex プロジェクト)としてインポートしてください。
  3. その後、Flash Builder 4 上でのビルド、実行が可能です。

コマンドライン

  1. 「準備(共通)」の手順を行います。
  2. Flex SDK をインストールし、Flex SDK インストールディレクトリ内の bin フォルダをPATH環境変数に指定してください。
  3. build.bat ファイルを実行してください。bin-debug ディレクトリ内にswfファイルが生成されます。

実行方法

インターネットブラウザ上で生成されたswfファイルを開きます。NFC Type 3 Tag をリーダー／ライタに置き、プログラム画面上の「Start」ボタンを押すと、読み込みと書き込みを実行します。

Flex SDK の fdb コマンドを使用することでデバッグメッセージ(trace)をコマンドラインに表示させることも可能です。
